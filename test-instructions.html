üêõ Troubleshooting

Extension Not Loading

  • Refresh the extension in chrome://extensions/
  • Check browser console for errors (F12)
  • Make sure you're on a LinkedIn job page

Analysis Failing

  • Ensure backend server is running on port 3000
  • Check if API_BASE_URL in popup.js is correct
  • Try with different resume file formats

History Not Working

  • Complete at least one analysis first
  • Check Chrome storage permissions
  • Try clearing browser data if issues persist
